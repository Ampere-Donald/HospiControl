# Maquettes / Designs

Dépose ici les maquettes visuelles (captures d'écran, exports Figma, images) qui
servent de référence pour l'UI de HospiControl.

Convention : quand une maquette concerne un écran précis, nomme le fichier d'après
l'écran (ex. `login.png`, `dashboard-medecin.png`, `consentement.png`).

Quand je dois implémenter un écran, je viens regarder les fichiers de ce dossier.
